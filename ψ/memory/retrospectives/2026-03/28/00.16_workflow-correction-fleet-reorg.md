## Lessons Learned

1. **`maw wake all --kill` doesn't remove sessions whose config was deleted** — must manually `tmux kill-session -t <old-name>` for orphaned sessions
2. **Fleet configs are symlinked** from `~/.config/maw/fleet/` → `/private/tmp/maw-js/fleet/`. When renaming, old symlinks break and new files may be copies instead of symlinks
3. **Athena added her own "My Role: DISPATCHER" section** to her CLAUDE.md after reading the broadcast — she understood the correction and self-reinforced it (good pattern)
4. **Batch agent updates work well with parallel sub-agents** — 16 agents × 3 edits = 48 edits completed in ~3 minutes using 3 parallel agents
