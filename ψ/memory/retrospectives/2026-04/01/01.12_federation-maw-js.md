## Lessons Learned

1. **maw-js cli.ts เป็น bundle** — source files แยกไม่ครบ ต้องสร้าง stubs จาก bundle ถึงจะ restart server ได้ ใช้เวลาเยอะกว่าที่คิด
2. **Plugin pattern ดีกว่า in-place mod** — extract ออกมาเป็น package ทำให้ clean กว่า ไม่ต้อง fight กับ missing deps ของ maw-js
3. **Dependency injection สำคัญ** — `createFederationRoutes(app, { listSessions, sendKeys })` ทำให้ plugin ไม่ depend on maw-js internals เลย
4. **Config แยกไฟล์** — ใช้ `federation.json` แทนที่จะแก้ `maw.config.json` ป้องกัน conflict กับ maw-js config อื่นๆ
