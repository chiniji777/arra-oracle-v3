## Lessons Learned

1. **ลด tabs = ใช้งานง่ายขึ้น** — จาก 13 tabs เหลือ 6 ทำให้ navigate เร็วขึ้นมาก
2. **ANSI escape sequences for arrow keys** — `\x1b[A` (up), `\x1b[B` (down) ส่งผ่าน WebSocket ไป tmux ได้ตรงๆ
3. **AppleScript iTerm2 tab creation** — `create tab with default profile` + `write text "tmux attach-session -t ..."` เปิด tmux session ใน tab ใหม่ได้สะดวก
4. **Fork workflow** — `gh repo fork` + `git remote set-url origin` แก้ปัญหา permission denied ได้เร็ว
