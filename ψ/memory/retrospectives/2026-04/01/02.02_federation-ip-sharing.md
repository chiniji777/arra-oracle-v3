## Lessons Learned

1. **Plugin dist ≠ host Vite** — เมื่อ plugin build ด้วย `bun build --external react` แต่ host ใช้ Vite + @vitejs/plugin-react, JSX runtime ไม่ตรงกัน (`jsxDEV` vs `jsx`) แก้ได้โดยให้ Vite compile source TSX ตรงๆ
2. **Copy source > import dist** สำหรับ frontend components ที่ใช้ JSX — backend code (pure TS) import จาก dist ได้ปกติ แต่ React components ควรให้ bundler ของ host compile
